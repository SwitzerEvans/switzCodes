Introduction to HTML - My First Step Into Web Development

📅 August 7, 2025

✍️ By Evans

Hey there! 👋

Welcome to my very first blog post on swizcodes. I'm super excited to start this journey of learning web development, and I thought the best way to stay motivated (and maybe help others) is by writing down everything I learn.

So, let's start right from the beginning — with HTML.

🤔 So… What is HTML?

When I first heard the word "HTML", it honestly sounded like one of those scary tech terms. But once I started learning it, I realized it's not that hard at all.

HTML stands for HyperText Markup Language — but don't let the fancy name fool you. It's just the basic language used to build websites. Every website you visit is using HTML behind the scenes.

Think of it like this:
If a website was a house, HTML would be the bricks and walls — it gives the site its structure.

My First HTML Code

<!DOCTYPE html>
<html>
  <head>
    <title>My First HTML Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

At first, all those tags and angle brackets looked confusing, but after a bit of practice, it started to make sense.

What Each Part Means:

  • <!DOCTYPE html> - Tells the browser, "Hey, this is an HTML5 page."
  • <html> - The root element of the page.
  • <head> - Stuff here isn't shown on the page, but it's still important.
  • <title> - The name that appears in the browser tab.
  • <body> - This is where everything on your webpage actually shows up — text, images, buttons, etc.
  • <h1> - A big heading. There are also <h2>, <h3>, etc.
  • <p> - A paragraph of text.

💬 Some Random Beginner Tips That Helped Me

  • Always close your tags (like <p></p>). Trust me, I forgot a lot at first 😅
  • Save your files with .html at the end, like index.html
  • Use a browser to open your file and see what you built
  • VS Code makes life easier as a beginner

🎯 Why I'm Loving This

Learning HTML gave me that “Whoa, I actually made this!” feeling. It's not flashy like animations or JavaScript (yet), but it's the foundation of everything on the web. And now I finally understand how websites are built from scratch.

This post is just the start. I'll be learning CSS and JavaScript soon, but for now, I'm focusing on getting my HTML basics solid.

👉 What's Next?

In my next blog post, I'll talk about the HTML structure and explain how all the parts of a webpage are organized. I'll try to break it down in a simple way, just like I'm learning it.

Thanks for stopping by and reading my first post. If you're also learning HTML, let's grow together! 🙌

Catch you in the next one!

— Evans 💻
