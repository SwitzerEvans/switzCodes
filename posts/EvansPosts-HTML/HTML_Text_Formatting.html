üè∑Ô∏è What Is Text Formatting?

Text formatting is how you style or emphasize parts of your text without using CSS. HTML gives you several simple tags to do this.

Here are the most common ones:


<b> - bold text  
<strong> - strong emphasis  
<i> - italic text  
<em> - emphasized text  
<u> - underlined text  
<mark> - highlighted text  
<small> - smaller text  
<del> - deleted text  
<ins> - inserted text  
<sub> - subscript  
<sup> - superscript

These tags may look similar, but some carry extra meaning for search engines and accessibility tools. Let's look closer üëá

üí™ Bold and Strong Text

To make text bold, use <b> or <strong>.


<p>This is <b>bold text</b>.</p>
<p>This is <strong>important text</strong>.</p>

üí° Difference:

  • <b> just makes the text bold visually.
    • ;
  • <strong> adds meaning ‚Äî it tells browsers that the text is important.

üìå Tip: Use <strong> when you want to emphasize meaning, not just looks.

‚ú® Italic and Emphasized Text

Italic text can show quotes, thoughts, or emphasize words.


<p>This is <i>italic text</i>.</p>
<p>This is <em>emphasized text</em>.</p>

üí° Difference:

  • <i> changes the appearance only.
  • <em> adds stress or importance ‚Äî great for accessibility.

üìå Tip: Use <em> when you want to stress a word's meaning.

üñäÔ∏è Underlining Text

If you want to underline something, use <u>.

<p>This is <small>small text</small>.</p>

üí° What They Do:

  • <mark> ‚Üí highlights text
  • <small> ‚Üí makes text smaller
  • <del> ‚Üí shows deleted content
  • <ins> ‚Üí shows newly added content

üß† Things I've Learned

  • <b> and <strong> both make text bold, but <strong> adds meaning
  • <i> and <em> both italicize text, but <em> adds emphasis
  • Use <u> carefully ‚Äî it looks like a link
  • Tags like <mark>, <small>, <del>, <ins>, <sub>, and <sup> are great for extra formatting details
